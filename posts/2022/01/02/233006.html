系统调优

Created2022-01-02|Updated2022-01-02
|Post Views:

默认情况下，RHEL8已经安装了tuned软件包，并启用了tuned.service。如未安装，则需要手动进行安装并启动、启用tuned.service。

[root@rhel8 ~]# yum -y install tuned

[root@rhel8 ~]# systemctl start tuned.service

[root@rhel8 ~]# systemctl enable tuned.service

tuned-adm命令可用于更改tuned守护进程的设置。tuned-adm命令可以查询当前设置、列出可用的配置文件、为系统推荐调优配置文件、直接更改配置文件或关闭优化

使用tuned-adm active命令来确定当前活动的调优配置文件

[root@localhost mnt]# tuned-adm active

Current active profile: virtual-guest

使用tuned-adm list命令可以列出所有可用的调优配置文件，包括内置的配置文件和系统管理员创建的自定义调优配置文件

[root@localhost mnt]# tuned-adm list

使用tuned-adm profile profilename命令可以将活动的配置文件切换为更符合系统当前调优要求的其他配置文件

[root@localhost mnt]# tuned-adm profile virtual-guest

使用tuned-adm recommend命令可以为系统推荐调优配置文件，该机制用于在安装后确定系统的默认配置文件

[root@localhost mnt]# tuned-adm recommend

virtual-guest

要恢复当前配置文件所做的设置更改，可以切换到另一个配置文件或停用tuned守护进程。

利用tuned-adm off命令关闭tuned调优活动

[root@localhost mnt]# tuned-adm off

[root@localhost mnt]# tuned-adm profile virtual-guest
