nmcli 网络管理

Created2021-12-09|Updated2021-12-09
|Post Views:

在RHEL 8中主机网络接口配置文件保存在**/etc/sysconfig/network-scripts**目录中。

手动方式配置网络时，需要配置的参数有：IP地址、子网掩码、网关、DNS地址及启用连接。

nmcli 命令配置网络

  1. 列出当前主机所有可用的网络连接：nmcli connection show

[root@localhost ~]# nmcli connection show

  1. 删除连接：nmcli connection delete 连接名称

[root@localhost ~]# nmcli connection delete ens160

  1. 添加网络连接：nmcli connection add

# con-name 连接名称 ifname 设备名称 type 类型 ethernet 以太网

[root@localhost ~]# nmcli connection add con-name ens160 ifname ens160 type ethernet

  1. 修改网络连接：nmcli connection modify 连接名称

（前提是NAME连接已经存在）

例：

IP：172.16.30.130/24 网关：172.16.30.1 DNS：202.103.24.68 启用连接

[root@localhost ~]# nmcli connection modify ens160 ipv4.addresses 172.16.30.130/24 ipv4.gateway 172.16.30.1 ipv4.dns 202.103.24.68 ipv4.method manual connection.autoconnect yes

# method：配置方式 manual：手动 connection.autoconnect：自动连接

  1. 激活连接：nmcli connection up 连接名称

[root@localhost ~]# nmcli connection up ens160

  1. 添加并配置网络连接：

[root@localhost ~]# nmcli connection add con-name ens160 ifname ens160 type ethernet ipv4.addresses 172.16.30.130/24 ipv4.gateway 172.16.30.1 ipv4.dns 202.103.24.68 ipv4.method manual connection.autoconnect yes

修改配置文件配置网络

  1. 查看ens160配置文件

[root@localhost ~]# cd /etc/sysconfig/network-scripts/

[root@localhost network-scripts]# ls

ifcfg-ens160

[root@localhost network-scripts]# vim ifcfg-ens160

修改如下配置：

BOOTPROTO=none 启动地址协议：none 手动 / dhcp 自动

ONBOOT=yes 是否连接 （激活）

IPADDR=172.16.30.130 IP地址

NETMASK=255.255.255.0或PREFIX=24 子网掩码

GATEWAY=172.16.30.1 默认网关

DNS1=202.103.24.68 dns地址

  1. 重新加载配置文件

[root@localhost network-scripts]# systemctl restart NetworkManager或nmcli connection reload ens160

  1. 激活ens160连接

[root@localhost network-scripts]# nmcli connection up ens160
