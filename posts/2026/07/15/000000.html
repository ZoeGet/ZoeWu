一、什么是远程仓库

远程仓库就是托管在其他位置的 Git 仓库，常见位置包括：

  • GitHub；
  • Gitee；
  • GitLab；
  • 公司内部 Git 服务器；
  • 自己搭建的 Git 服务。

本地仓库负责保存本机代码和提交记录，远程仓库负责同步、备份和多人协作。

十二、常用命令汇总

命令 作用
git remote 查看远程仓库名称
git remote -v 查看远程仓库名称和地址
git remote add origin <url> 添加远程仓库
git remote show origin 查看远程仓库详细信息
git remote set-url origin <url> 修改远程仓库地址
git remote rename origin github 重命名远程仓库
git remote remove origin 删除远程仓库关联
git push -u origin main 第一次推送并建立跟踪关系
git push 后续直接推送
